# MiSTercade V1-2025 Mappings (No Menu Combo)

V1-2025 No-Menu mapping database for the [MiSTercade](https://misteraddons.com/products/mistercade).

This variant has the **menu combo (Down + Start) disabled** so it cannot be accidentally triggered during gameplay.

## How to Integrate into MiSTer Downloader

Add the following to the end of `downloader.ini` on your MiSTer SD card:

```ini
[misteraddons/mistercade_v1_2025_nomenu_mappings]
db_url = https://raw.githubusercontent.com/misteraddons/mistercade_v1_2025_nomenu_mappings/db/db.json.zip
```

Then run **downloader** or **update_all**.
