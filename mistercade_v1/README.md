# MiSTercade V1

This folder contains files for MiSTercade V1.

## Folders

- [`mappings/`](mappings/) contains Downloader `.ini` choices for MiSTer `.map` files.
- Firmware update files and flashing instructions are published under [`../mistercade_v1_2025/firmware/`](../mistercade_v1_2025/firmware/).

Note: MiSTercade V1 non-2025 is legacy/deprecated due to Jotego framework changes. Use MiSTercade V1 2025 whenever possible.
