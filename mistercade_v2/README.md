# MiSTercade V2

This folder contains files for MiSTercade V2.

## Folders

- [`mappings/`](mappings/) contains Downloader `.ini` choices for MiSTer `.map` files.
- [`gamecontrollerdb/`](gamecontrollerdb/) contains experimental Downloader `.ini` choices for `gamecontrollerdb_user.txt`.
- [`firmware/`](firmware/) contains firmware binaries, flashing instructions, and checksums.
